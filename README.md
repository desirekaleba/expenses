# expenses

An expenses tracker app built using Flutter.

## Expense Model
```
{
    id: String,
    title: String,
    amount: Double,
    date: DateTime
}
```

## Features

* View expenses
* Add expense
* Delete expense

## Dependencies
* [Flutter](https://flutter.dev)
* intl
* cupertino_icons

## How to run the app

Clone this repo and run:
```
flutter run
```
